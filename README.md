#  Retail Sales Data Analysis using SQL

##  Project Overview
This project focuses on analyzing a messy retail sales dataset using SQL.  
The dataset contains inconsistencies, duplicate records, and formatting issues.

## 🧹 Data Cleaning
- Identified duplicate transactions  
- Detected invalid values (zero/negative entries)  
- Checked missing values  

## 📊 Analysis Performed
- Total sales by category  
- Top 5 revenue-generating products  
- Categories performing above average  
- Monthly sales trends  
- Top customers by spending  
- Payment method analysis  

## 💡 Key Insights
- A small number of products generate most of the revenue  
- Customer spending is concentrated among top buyers  
- Sales show clear monthly trends  
- Payment methods impact overall revenue distribution  

## 🛠 Tools Used
- MySQL Workbench  
- SQL  

## 📂 Project Files
- analysis.sql → SQL queries  
- retail_store_sales.csv → dataset  

##  Conclusion
This project demonstrates how raw and messy data can be cleaned and transformed into meaningful business insights using SQL.
